### 🚀 프로젝트 소개

**AuraTalk**은 랜덤 채팅 플랫폼으로, 관심사가 비슷한 사람들과 연결되어 친구를 만들고 점차 친구의 범위를 넓혀갈 수 있는 환경을 제공합니다.

**주요 특징**
- 실시간 메시징
- 음성 통화
- 랜덤 매칭
- 그룹 채팅
- 관심사 설정
- 파일 공유
- 친구 수락/거절/차단
